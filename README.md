# Teste de Estágio 

## Visão Geral
Este projeto foi desenvolvido como parte do processo seletivo
para a vaga de estágio.
O foco foi demonstrar capacidade de organização, tomada de decisão
técnica e documentação clara.

## Observação Importante
Devido a limitações de hardware, o projeto foi estruturado priorizando
clareza, modularização e documentação.
Os códigos apresentados consideram execução local em Python,
porém não foram executados neste ambiente.

## Tecnologias Propostas
- Python 3
- Pandas
- PostgreSQL
- SQL

## Estrutura do Projeto
- etapa1_api_ans: Integração com API pública da ANS
- etapa2_validacao: Validação, enriquecimento e agregação
- etapa3_sql: Scripts SQL e análises
- dados: Arquivos CSV gerados

## Decisões Técnicas
- Processamento incremental para reduzir uso de memória
- Estrutura modular para facilitar manutenção
- Não utilização de banco local por simplicidade e limitações do ambiente

## Limitações Conhecidas
- Código não executado neste ambiente
- API e frontend não implementados
- Normalização simplificada dos dados

## Próximos Passos
- Execução em ambiente com mais recursos
- Implementação da API em FastAPI
- Desenvolvimento da interface web em Vue.js
